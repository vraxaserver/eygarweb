# User actions
1. **User can signup**
2. **User can login**
3. **User can become a host**
4. User can become a vendor
5. User can search property by location
6. User can filter property by checkin, checkout, amenities etc
7. User can Update profile settings (Name, Avatar)
8. User can become Host and go to the Host dashboard and perform following actions:
    1. View recent bookings
    2. Add property
    3. View property listing
    4. Update/Delete property listing
    5. View Active guest listing
    6. View Experiences
    7. Add/Update/Delete Experiences
    8. View the analytics
9. User can Go to Guest Dashboard
    1. View Upcoming bookings
    2. View history of bookings
